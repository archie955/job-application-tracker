# Full Stack Job Application Tracker

This is a full-stack job application tracker with the ability to track jobs and their corresponding assessments and stages.

## Contents
1) [Technology Stack](##Stack)
2) [Usage](##Usage)
3) [Roadmap](##Roadmap)


## Stack
Uses PostgreSQL, Python, JavaScript, HTML, CSS, Docker, GitHub Actions

**Database:** PostgreSQL

**Backend:** FastAPI, Pytest, SQLAlchemy, Pydantic, Alembic

**Frontend:** React

Containerised using Docker. GitHub Actions used to automate testing, building, and deployment to remote server.

## Usage

Website is live at [Domain here when purchased](). 

Functionality to create account, add jobs and assessments, continuously update them keeping a history of all applications, the results, all assessments that both have already been completed and are yet to be done, with deadlines so you never miss one.
